# 🎬 Movie Watchlist

A responsive web application that allows users to search for movies using the OMDB API and save them to a personal watchlist. Built with vanilla JavaScript, HTML, and CSS.

## ✨ Features

### 🔍 Movie Search

- Search for any movie using the OMDB API
- Each result shows:
  - Movie poster with fallback placeholder
  - Title and release year
  - IMDb rating
  - Runtime
  - Genre
  - Plot description

### 📖 Plot Description

- Long plot descriptions are truncated
- "Read more" button expands to show full plot
- Smooth expand/collapse functionality

### 📋 Watchlist Management

- Add movies to personal watchlist
- Remove movies from watchlist
- Dedicated watchlist page to view saved movies

### ♿ Accessibility

- ARIA live regions for screen readers
- Semantic HTML structure
- Keyboard navigable interface
- Proper focus management

## 🛠️ Technologies Used

- HTML5 - Semantic markup
- CSS3 - Flexbox, Grid, custom properties
- JavaScript (ES6+)
- OMDB API - Movie database integration
- LocalStorage API - Persistent watchlist storage
